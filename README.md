# Blocky Multiplayer Demo

Multiplayer demo built in Unity with a TypeScript server.

## Features
- Room join by code
- Multiplayer character sync
- Walk / jump animations
- Sit animation
- Character skin switching

## Tech Stack
- Unity Client
- TypeScript Node.js Server

## How to Run

### Client
Open UnityClient folder in Unity and press Play.

### Server
cd Server
npm install
npm run dev
